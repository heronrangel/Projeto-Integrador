# Lavanderia Campo Grande - Sistema de Gestão

## Sobre o Projeto

O **Lavanderia Campo Grande** é um sistema desenvolvido como Projeto Integrador do curso de **Análise e Desenvolvimento de Sistemas** da Universidade La Salle.

O objetivo do projeto foi criar uma solução tecnológica capaz de auxiliar na gestão operacional da empresa parceira, centralizando informações e automatizando processos que anteriormente eram realizados manualmente.

## Solução Desenvolvida

O sistema foi desenvolvido para:

* Controle de pedidos;
* Gestão de produtos e serviços;
* Controle de insumos;
* Organização das informações da empresa;
* Automatização de processos operacionais;
* Melhor acompanhamento das atividades da lavanderia.

## Tecnologias Utilizadas

* C#
* .NET
* Windows Forms
* SQLite
